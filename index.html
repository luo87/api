Twip 使用说明

什么是 Twip？

Twip 是一个运行在 LAMP 主机上的 PHP 程序，用于绕过GFW的限制在各种Twitter客户端里访问Twitter。

什么是 T 模式？

T模式 ( Transparent ) 会透明转发客户端的HTTP请求，能完整保留客户端的Source信息。

什么是 O 模式？

O模式 ( Override ) 会使用自定义OAuth Consumer Key/Secret重新对客户端HTTP请求进行签名，无法保留客户端Source信息。
